Selenium Web 测试
1. 首先需要安装对应的库pip install --pre --upgrade robotframework-seleniumlibrary
2. 然后下载 chromedriver，下载地址为：https://npm.taobao.org/mirrors/chromedriver
下在文件根据系统而定：我的是chromedriver_win32.zip

要运行成功此用例：
1. 需要将下载chromedriver.exe 放在robot 对应的python 的同级目录下：
比如：我的python在目录：
C:\Users\wrm\AppData\Local\Programs\Python\Python37-32
就把chromedriver.exe 也放在这个目录下
